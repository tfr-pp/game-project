# SkyLink

Tiny game project made with monogame

The game isnâ€™t finished yet. We still have to work on a settings screen, a death screen, more levels, etc. We preferred to focus on the physics and reliability to make the game playable rather than on visual effects and menus.

## Requirements

- Monogame framework
- .NET 9.0 >
- An IDE preferably VS Code or JetBrains Rider

## Install

Go to jeu.DesktopGL and run:

- `dotnet restore`
- `dotnet build`
- `dotnet run`
